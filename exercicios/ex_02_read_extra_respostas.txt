Ordene pela data de inclusão.
Ordene pelas descrições.
Ordene pela quantidade de caracteres das descrições.
Procure somente registros aonde o dia da data de inclusão seja igual a 01.

db.tasks.aggregate(
  [
    {
      $project: {
        "descricao": 1,
        "length": { $strLenCP: "$descricao" }
      }
    }
  ]
).pretty()

db.tasks.aggregate(
  [
    {
      $project: {
        "descricao": 1,
        "length": { $strLenCP: "$descricao" }
      }
    }, 
	{ $sort: {length : 1} }
  ]
).pretty()
